# Avatar Gallery Collections

A premium collection of high-quality avatars, organized by category, designed for the Avatar Gallery XenForo addon.

## Collections Included

| Category       | Description                                  | Count |
| -------------- | -------------------------------------------- | ----- |
| **GTA**        | Grand Theft Auto stylized character artwork. | ~10   |
| **Pixel Art**  | Retro-style pixel art heads and icons.       | ~112  |
| **Zombatar**   | Fun, spooky zombie characters.               | ~27   |
| **Characters** | Diverse general character portraits.         | ~50   |
| **Cryptopunk** | CryptoPunk NFT style avatars.                | ~125  |

## Installation

1. Download the addon from Xenforo https://xenforo.com/community/resources/avatar-gallery.10211/
2. Download this repository contents.
3. Upload the folders (`GTA`, `PixelArt`, etc.) into your XenForo directory:
   `data/gallery_avatars/Global/`
4. Done! The new tabs will automatically appear in your Avatar Gallery.

## Specifications

- **Format**: High-quality PNG/JPG
- **Dimensions**: Optimized for 200x200px
- **Total Size**: Lightweight (~12MB total)
